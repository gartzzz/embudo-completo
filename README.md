# 🎯 Embudo Completo

Skill para Claude que te lleva **de cero a un call funnel de alto ticket desplegado en Vercel**: landing (Astro), VSL, aplicación con agenda integrada, página de confirmación que convierte y página de descalificados. Con sistema de diseño propio anti-vibecoding y copy visceral.

## Qué hace

Trabaja por fases con puertas de aprobación — nunca te vuelca todo de golpe:

1. **Intake** — te pide tus documentos (oferta, ICP, criterios de cualificación, calendario), tu brand kit (logo, colores, tipografías, fotos reales, referencias) y hace un cuestionario de brand voice.
2. **Arquitectura** — diseña el funnel: landing + aplicación con agenda en UN paso (Typeform×Calendly), confirmación con pixel, página DQ sin pixel.
3. **Copy** — escribe todo el copy con la estructura El Círculo y direct response clásico, en TU voz.
4. **Sistema de diseño** — genera un MASTER de diseño (paleta, tipografía, estilo, motion, elemento firma) que huye del look genérico de IA.
5. **Construcción** — genera el proyecto Astro completo, móvil primero, con tokens de diseño.
6. **Despliegue** — te guía para publicarlo en Vercel por MCP (sin terminal) o CLI (copy-paste), y verifica el pixel en vivo.
7. **Post-lanzamiento** — plan de contenido "Hammer Them", benchmarks y modelo financiero para escalar.

## Instalación

### Claude (web/desktop — claude.ai)

1. Descarga este repo como ZIP (Code → Download ZIP) o comprime la carpeta que contiene `SKILL.md`.
2. En Claude: **Ajustes → Capacidades → Skills → Subir skill** y sube el ZIP.
3. En un chat nuevo, escribe: *"crea mi funnel"* o *"embudo completo"*.

> Alternativa sin skills: pega el contenido de `SKILL.md` como instrucciones de un Proyecto.

### Claude Code

```bash
git clone https://github.com/gartzzz/embudo-completo ~/.claude/skills/embudo-completo
```

Y en cualquier sesión: `/embudo-completo` o simplemente pide "crea mi funnel".

## Qué necesitas tener a mano

- Tu oferta (qué vendes, precio, qué incluye) — ideal para tickets de $3K+
- Quién es tu cliente ideal y qué lo cualifica/descalifica
- Link de Calendly o Cal.com
- Logo, colores y fotos reales de tu marca
- Testimonios en captura (si los tienes)
- Cuenta gratuita de Vercel

La skill no bloquea si te falta algo: lo reconstruye contigo a base de preguntas.

## Créditos y fuentes

- Blueprint de call funnels, VSL y confirmation pages basado en los frameworks públicos de **Jeremy Haynes** (Megalodon Marketing).
- Metodología de copy **Clientbubble / El Círculo**, con munición de direct response clásico (Gary Halbert).
- Dirección de arte inspirada en **ui-ux-pro-max** (sistema de diseño + reglas anti-vibecoding).
